## Introduction to React

React is a JavaScript library for building user interfaces, maintained by Facebook and a community of individual developers and companies.

### Key Concepts Covered

#### 1. Components
- Functional components vs Class components
- Props and state
- Component composition

#### 2. JSX Syntax
- Writing HTML-like syntax in JavaScript
- Using expressions in JSX with curly braces
- Conditional rendering

#### 3. Hooks
- useState hook for state management
- useEffect hook for side effects
- Custom hooks

### Important Notes

Always remember to:
- Break down UI into components
- Keep components small and focused
- Use props to pass data down
- Lift state up when needed
